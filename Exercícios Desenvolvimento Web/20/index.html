Elementos texto

Elementos Containers

Container é onde varios outros elementos estarão contidos dentro do mesmo, para declara um container utilizamos a tag < div class="container"> </div>

Elementos Formulário

A tag utilizada para declara um formulário é "< form >< /form>"

Elementos multimedia
